🏨 Hotel Reservation System
A desktop-based **Hotel Reservation System** developed using **Java AWT** and **MySQL**. The application simplifies hotel management by providing an intuitive interface for managing customers, rooms, reservations, and payments.

📌 Overview

This project was developed as part of my **Java & DSA Placement Training** at **Blue Birds**. It demonstrates the practical application of Java programming, Object-Oriented Programming (OOP), GUI development, JDBC connectivity, and MySQL database management.

✨ Features
- 🔐 Admin Login
- 👤 Customer Registration
- 🛏️ Room Management
- 📅 Room Booking
- 📋 Reservation Management
- 💳 Payment Management
- 🗄️ MySQL Database Integration

🛠️ Tech Stack
- **Java**
- **Java AWT**
- **JDBC**
- **MySQL**
- **IntelliJ IDEA**
- **Git**
- **GitHub**

⚙️ Database Setup
1. Create a MySQL database.
2. Import the SQL file located in the `database` folder.
3. Update the database credentials in the JDBC connection class.

## 🚀 How to Run
1. Clone the repository.
```bash
git clone https://github.com/your-username/HotelReservationSystem.git
```
2. Open the project in IntelliJ IDEA.
3. Add the MySQL Connector (JDBC Driver).
4. Import the database.
5. Update the database credentials.
6. Run the application.
